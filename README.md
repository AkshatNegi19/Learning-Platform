# 📖 Overview

AI Learning Platform is a modern learning management application built using the MERN Stack.

The platform enables users to search for any technology, discover curated learning resources, test their understanding through quizzes, and monitor their progress using an interactive dashboard.

The primary goal of this project is to create a centralized platform that combines learning, assessment, and progress tracking in one place.

## 📝 Quiz System

Each technology includes quizzes to evaluate learning.

Features include

- Instant Results
- Score Calculation
- Performance Analysis
- Correct Answer Review

Every user has a personalized dashboard displaying
